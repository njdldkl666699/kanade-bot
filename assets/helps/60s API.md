# 60s API

## 🕘 周期资讯

### `/每天60秒读懂世界`

别名：`/60s` `/60秒` `/日报`

获取每日60秒读懂世界资讯内容。

### `/AI资讯快报 [日期]`

别名：`/ai新闻` `/AI资讯` `/ai_news` `/ai-news`

参数：`日期`可选，默认当天，格式：`2025-11-11`

提供全面的AI、大模型等领域的最新AI资讯。新闻不是每天都有！

### `/Epic`

别名：`/epic` `/epic游戏`

获取 Epic Games 每周免费游戏列表

### `/实时IT资讯`

别名： `/it新闻` `/IT资讯` `/it_news` `/it-news`

获取实时IT资讯，内容来自IT之家，返回最新的10条。

## 🍱 实用功能

### `/实时天气 [城市名称]`

别名：`/天气` `/weather`

参数：`城市名称`可选，默认`北京`。

获取指定城市的实时天气信息。

### `/天气预报 [城市名称] [天数]`

别名：`/weather_forecast`

参数：

- `城市名称`可选，默认`北京`；
- `天数`可选，默认`7`。

获取指定城市未来几天的天气预报信息。

### `/翻译 <文本> [目标语言] [源语言]`

别名：`/fanyi` `/translate`

参数：

- `文本`必填，要翻译的文本内容；
- `目标语言`可选，默认`auto`，源语言为中文则目标为英文，否则目标为中文；支持模糊匹配语言名称和精确匹配语言代码。
- `源语言`可选，默认`auto`，即自动检测；支持模糊匹配语言名称和精确匹配语言代码。

数据来源于有道翻译，与其网页端同步。

### `/摸鱼日报`

别名：`/摸鱼` `/moyu`

获取摸鱼日报内容。

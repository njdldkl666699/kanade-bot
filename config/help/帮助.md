# 帮助

### `/帮助 [模块名]`

别名：`/help` `/?` `/帮助文档`

参数：`模块名`可选，指定要查看的模块名称，如不指定则返回此帮助文档。

获取帮助信息，返回帮助文档图片。对于Console适配器，直接返回帮助文档Markdown文本。

### `/状态`

别名：`/status`

`wyf9/nonebot-plugin-picstatus-ng`插件提供的图片版运行状态。仅支持OneBot v11适配器。

### 上线通知

Bot上线时会向配置的群聊发送通知消息，告知用户Bot已上线。

### 掉线通知

Bot掉线时会通过Server酱Turbo推送通知到微信，告知部署者Bot已掉线。

### `/execute <Shell命令>`

别名：`/exec`

权限：`SUPERUSER` 且 仅OneBot v11私聊

参数：`Shell命令`必填，要执行的Shell命令。

在服务器上执行指定的Shell命令，并返回命令的输出结果。仅适用于OneBot v11适配器的私聊消息，且需要超级管理员权限。

### 新群员欢迎

QQ群中有新成员加入时，Bot会发送欢迎消息。欢迎消息可以在配置文件中自定义，支持文本和图片。

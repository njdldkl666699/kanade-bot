# 宵崎奏Bot帮助文档-{{project_version}}

> 1. `/`开头的为命令，不需要`@宵崎奏`，尽量避免`@宵崎奏`的同时使用`/`命令，以免造成冲突。
> 2. `<>`包围的参数必填，`[]`包围的参数可选。
> 3. 命令与参数之间的空格是可选的。
> 4. 标星\*的命令表示需要超级管理员（`SUPERUSER`）权限。
> 5. 模块名即帮助文档中一级标题的内容；不指定模块名则返回本文档。

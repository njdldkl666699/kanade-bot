# 今天吃什么

来自插件`nonebot-plugin-whateat-pic`。

## `/今天吃什么` `/今天喝什么`

匹配的正则表达式：

```
[今明后]?
[天日]?
(?:早|中|晚)?
(?:上|午|餐|饭|夜宵|宵夜|早|晚)?
[吃喝]
(?:什么|啥|点啥)
```

随机发送食物图片，或者饮品图片。

## `/全部菜单 <img_type>`

别名：`/查看菜单` `/查看菜品`

参数：`img_type`必填，`菜单` `菜品`表示食物菜单，`饮料` `饮品`表示饮品菜单。如果不填，Bot会提示用户继续填写正确的参数，如果提供了错误的参数，Bot会提示用户参数错误并需要重新发送指令。

## `/添加菜单 <name> <img_type> <img>`

权限：`SUPERUSER`

添加自定义菜单。

## `/删除菜单 <name> <img_type>`

权限：`SUPERUSER`

删除自定义菜单。

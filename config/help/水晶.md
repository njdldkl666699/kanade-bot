# 水晶

### `/签到`

别名：`/checkin`, `/check_in`, `/打卡`, `/打卡啦摩托`

每天可以签到一次，获得随机数量的水晶，部分功能需要消耗水晶。

### `/我的水晶`

别名：`/mycrystal`, `/my_crystal`, `/查看水晶`

查看当前拥有的水晶数量。

### `/查看命令水晶消耗`

别名：`/查看命令消耗`, `/handler_consumes`, `/list_handler_consumes`

列出所有需要消耗水晶的功能及消耗数量。

### `/水晶排行榜`

别名：`/水晶排名`, `/crystal_ranking`

查看水晶排行榜，显示拥有水晶数量排名前十的用户。
